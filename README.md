Indian Road Accident Analysis (2018–2023)

Project Overview 
- This project analyzes road accident data from across India (2018–2023) to uncover patterns, severity trends, and key risk factors. Using Python for data cleaning and Excel for interactive dashboards, the project aims to visualize the who, where, and why behind road accidents, helping identify actionable safety insights.

Tools & Technologies
- Python (Jupyter, Pandas, NumPy) – Data cleaning, transformation, and export
- Microsoft Excel – Dashboards, KPIs, charts, and slicers
- Dataset Source – Kaggle: Indian Road Accidents (2018–2023)

Key Insights
- The analysis highlights yearly accident trends, severity distribution, state-wise accident hotspots, and the impact of weather and road conditions.

How to Use
- Download the cleaned_data.xlsx file.
- Open it in Microsoft Excel.
- Click Enable Editing to interact with slicers and dashboards.
- Explore visuals across different states, years, and conditions.

Project Outcome
- This project demonstrates end-to-end data analysis skills, from raw data cleaning to interactive visualization.
